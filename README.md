<h1> Dino Rush </h1>
Reimagination of the Endless Runner Dino Game built into Google Chrome.
<br>
<br>
<pre>
Space Bar - Jump
Enter - Play/Pause
</pre>

<h2> Instructions </h1>
<ol>
    <li> 
        Clone this repository
        <code>git clone https://github.com/aditdesai/dino-rush.git</code>
    </li>
    <li>
        Change directory
        <code>cd dino-rush</code>
    </li>
    <li>
        Run main.py
        <code>python main.py</code>
    </li>
</ol>
